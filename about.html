О квесте

Что такое квест?

File

Edit

View

Квест — сюжетная игра, где вы управляете главным героем, идете к выполнению основной задачи по сюжету и попутно разгадываете различные головоломки, выполняете интерактивные задания, находя и используя различные предметы.

В чём задача веб-безопасности?

File

Edit

View

Search

Terminal

Для эффективной безопасности веб-сайта необходимо уделять особое внимание к разработке всего веб-сайта: к вашему веб-приложению, конфигурации веб-сервера, а так же кода на стороне клиента.

Цель веб-безопасности заключается в изучении способов защиты веб-сайтов от несанкционированного доступа, использования, изменения, уничтожения или нарушения работы.

Угрозы безопасности

Edit

View

В этом разделе перечислены некоторые из наиболее распространённых угроз веб-сайта:

  • Межсайтовый скриптинг (XSS)
  • Denial of Service (DoS)
  • SQL injection
  • Недостатки системы аутентификации
  • Небезопасная конфигурация
  • LFI/RFI
  • SSI Injection

Безопасность веб-приложений

File

Edit

View

Уязвимости веб-приложений возникают тогда, когда разработчики добавляют небезопасный код в веб-приложение. Это может происходить как на этапе разработки, так и на этапе доработки или исправления найденных ранее уязвимостей. Недостатки часто классифицируются по степени критичности и их распространенности
